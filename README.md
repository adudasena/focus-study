# 📚 Focus Study

A full-stack solution to manage academic routines with advanced security features and personalized dashboards.

## 🌟 Key Features
* **Role-Based Access Control (RBAC):** Specific permissions for different user levels.
* **Data Isolation:** Ensures users only access their own records.
* **Admin Dashboard:** Real-time statistics and user management.
* **Responsive UI:** Built with Vue.js for a seamless experience.

## 🛠️ Technology Stack
* **Backend:** Java 17, Spring Boot, Spring Security (JWT)
* **Frontend:** Vue.js, CSS3
* **Database:** PostgreSQL
* **DevOps:** Docker & Docker Compose
